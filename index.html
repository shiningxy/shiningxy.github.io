考研全攻略

发表于 2020-10-29 更新于 2020-10-30 分类于 考研

前言

考研经验贴漫天皆是。兼听则明，偏信则暗。不要单信某一篇经验贴，要广泛搜集信息，结合自身情况，制定一套适合自己的考研计划才是良策。

基本概况扫盲

学硕和专硕的区别

  1. 培养方向不同，学术型硕士教育以培养教学和科研人才为主，授予学位的类型主要是学术型学位，专业型硕士是国家为了克服学术性硕士的不足而新增的一种硕士，培养的是市场紧缺的应用型人才。学硕一般三年制，专硕一般两年或两年半制；（学硕偏学术，专硕偏就业）

  2. 招生考试不同，基本分数一数二 英语一英语二区别；

  3. 院校招生名额不同，调剂有差异。学硕更方便调剂，专硕名额紧张，调剂有限制，更加困难。

考研有哪些专业

13大学科门类：哲学、经济学、法学、教育学、文学、工学、军事学、历史学、农学、管理学、艺术学、理学、医学

全国第四轮学科评估结果公布将学科分为六个：人文社科类、理学、工学、农学、医学、管理学、艺术学。对各个院校各学科进行了等级评估，可用于报考参考。

目标院校的选取

  1. 以后想在哪座城市发展（地域）

  2. 目标学校的社会认可度（学校）

  3. 学校该专业的强弱（专业）

考试科目

理工科通常考试科目为：

  • 政治(100分)
  • 英语(100分)
  • 数学(150分)
  • 专业课一(150分)

如果不考数学（法学等），考试科目为：

  • 政治(100分)
  • 英语(100分)
  • 专业课一(150分)
  • 专业课二(150分)

政治为统考科目全国一张卷

英语分为英语一和英语二，具体考试难度相差不大，也与当年具体情况有关。

数学分为数学一，数学二，数学三。数学二不考概率论。

专业课

分为统考专业课和自命题专业课

统考专业课，全国一张卷，题型分值往年真题都很容易找到。
自命题专业课，学校自行出题，一个学校一套卷，需要去学校买或者在咸鱼等平台买。自命题专业课的往年真题十分重要，往往都是往年原题再现。

考研学习复习时间表参考

准备阶段：2020年9月-12月搜集信息，筛选专业和学校，了解学习专业课内容。 基础阶段：2021年1月-3月打好数学和英语基础，注重单词和公式定理。 夯实基础阶段：2021年4月-5月第一轮基础复习，结合题量，继续夯实基础 提高阶段：2021年6月-8月第二轮强化复习，提升难度，谨记真题的重要性，多刷真题，8月份开始政治学习 强化阶段：2021年9月-10月继续强化复习，关注预报名，正式报名 冲刺阶段：2021年11月-12月第三轮冲刺复习，不要忘记真题的重要性。 临考阶段：2021年12月打印准考证，考研初试。 # 考研名师推荐 ## 个人推荐 每个名师就像一个门派，各门各派都有机会在华山论剑中取胜 每个名师的授课风格，知识体系，所用书籍都不相同 我的建议是：认准一个名师，跟到底，后期冲刺阶段押题可以多门派“偷学”。 数学：汤家凤 英语：唐迟 政治：徐涛 ## 网络推荐 ### 英语 **网课推荐：** 阅读：唐迟 长难句分析：田静 写作：王江涛 **资料推荐：** 1.词汇书：红宝书/恋练有词 2.真题详解：张剑黄皮书 3.作文：王江涛的考研英语高分写作 4.作文练字：王江涛的考研英语高分写作字帖 ### 数学 **网课推荐：** 高数：汤家凤基础班+张宇强化班 线代：李永乐 概率：王式安 **资料推荐：** 1.教材 2.高数18讲 3.李永乐线性代数辅导讲义 4.王式安概率论与数理统计辅导讲义 5.张宇1000题 6.张宇真题大全解 7.张宇8套卷 8.合工大5套卷 ## 政治 **网课推荐：** 徐涛强化班+刷题班 **资料推荐：** 1.肖秀荣4套卷 2.肖秀荣8套卷 3.肖秀荣精讲精练/徐涛核心考案 4.肖秀荣1000题 5.徐涛20题 6.徐涛8套卷（只做选择题部分） # 数学高分心得（转载） # 1月到3月 这个阶段其实是属于一个最基础的阶段，如果基础不错，这个阶段会很轻松。主要以课本为主。主要把基本概念，基本定理以及它的适用条件搞清楚，这些弄不清楚，会是很大的问题，后面会越学越糊涂。课后习题只把章测试做一下，后面小节测试就没必要了。推荐同济七版高数＋同济六版线代＋浙大四版概率论。

4月到7月

前期不要贪图题量和难度，重在夯实基础，可以看李永乐的复习全书同时做基础篇的习题，张宇1000题或者汤家凤的1800都是可以的。基础不是很好可以看看他们的课程配合习题集一起进行。基础特别不错可以做李永乐的660，660难度要大一些，里面很多都是考研中等难度的题。

7月到10月

这属于强化阶段，1800的强化篇和1000题提高可以开始做了，二刷李永乐复习全书。如果前面做了1800或者1000，可以这个时候做660，1800和1000题只做一本就可以，但李永乐的660必做。拿下660之前选填没啥大问题。如果觉得自己前面没有打好基础，可以看一些老师的辅导视频。

10月到12月

这个时候可以开始进行真题的训练了，2003年以前的真题当练习题做就可以。03年到20年的严格按照3小时时间来，准确地点说，应该在2个半小时内完成。真题是有规律的，大家要多加总结，就会发现可能10年前考过的题型和知识点会以相似的方式出现，大家一定要重视真题，利用好真题。可以做适当的模拟题，推荐李林的6套卷和4套卷，风格比较贴近真题，也有可能在考场上帮到大家。李林玄学押题每年必中，临近考试前会在全国各地开押题班，但也会有录音流出，找到押题笔记和课堂录音即可

12月到考前

最后三周，大家都在忙于背政治和英语作文，但是数学得保证至少4个小时学习时间。一定保持手感，计算能力在后期尤为重要。考场上要稳住心态，题目难大家都难，谁的心态不崩，谁就是最后胜利者。

2020/10/9 会议纪要

会议概况

会议时间：2020/10/9
会议地点：商船学院A座321
会议主持：刘轶华老师
参会人员：马钰 汪婷 王翔宇 周建 张静燕
会议记录：王翔宇

会议概要

刘老师在会议中指出，10月份开学相比于正常9月份开学，时间更紧。

这就要求新一届研究生加快从本科生到研究生的身份转变，从心理层面上认识到自己已经是研究生，从学术角度上要习惯阅读文献，论文写作。

刘老师的研究生们，平时的学术研究/学习场所主要以图书馆为主。

生活上合理安排，心态放松愉悦，早睡早起，良好的作息和健康的身体才是今后拼搏、革命的本钱。

争取国家奖学金（尽量争取特等奖）

问题梳理

专业选修课要求

  1. 根据自己的知识学习需求，选择专业选修课。
  2. 可以选择刘轶华老师任教的《交通流理论》。

小论文要求

  1. 学硕：至少一篇SCI文章，分区尽可能高（1区最高，4区最低）。一篇EI文章也可以，不过效果会大打折扣。努力发表SCI高区。
  2. 专硕：要求同学硕，若发表上海海事大学学报也勉强认可。
  3. 版费资助：资助D类以上期刊，不资助D类期刊（如：上海海事大学学报）

专利要求

  1. 学硕专硕统一要求：一个发明专利
  2. 取得研究成果之后，不要先发表文章，应先申请专利保护，防止专利被先发表的文章影响了新颖性而不被授权。

署名规矩

  1. 文章核心思想是学生提出：学生是第一作者，刘老师是通讯作者。
  2. 文章核心思想是刘老师提出，而学生负责操刀撰写：刘老师是第一作者，学生是第二作者。
    上述两种方式，对学生本人加分相同。

形势分析和下步计划

《船舶交通工程学》

  1. 下周组内共同任务：搜集全国与《船舶交通工程学》相关的书籍，归纳整理所有书籍，总结出一个适合本科生或研究生的目录/章节，为此书后续的出版做先行工作。预计耗时一周。

学习编程语言/工具

  1. Python
  2. R

上述语言中选择一种学习，可以选择胡志华老师的Python公开课,也在网络上广泛地搜罗资料。

有目的性地敲代码，每实现一个功能或完成一个项目，都会给予自己正向激励，相较于机械式地学习更有动力。

R语言是用于统计分析、绘图的语言和操作环境，它是一个用于统计计算和统计制图的优秀工具，常用于学术交流会议。

  1. Matlab

Matlab中Simulink是一个用来对动态系统进行建模、仿真和分析的软件包。在船舶建模与仿真领域也有应用，值得学习。

研究生课程论文

  1. 研究生课程结课考核方式大多以提交结课论文形式。
  2. 大量阅读课程相关方向的文献。
  3. 结课论文如果写得好，可以加以修饰润色进行发表。一举多得!

优化方案1：提前return，去掉不必要的else

如果 if/else 代码块包含 return 语句，可以考虑通过提前 return ，把多余的 else 干掉，使代码更加优雅。
